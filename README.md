# Product Cards

A simple product cards application.

## Description

A simple shopping cards app built using HTML, CSS, JavaScrip, and the [Fake Store API](https://fakestoreapi.com/).


## Getting Started

### Executing the program
Clone the repository:
```
git clone https://github.com/Lirmortoch/shopping-cards.git
```
Run the server (e.g. [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)).
